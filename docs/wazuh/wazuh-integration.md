# Wazuh Integration Plan

## Objective

Integrate one detection scenario from this Detection-as-Code Lab into Wazuh and validate it with a real alert.

## First use case

The first integration will focus on failed SSH login attempts.

## Why this use case?

SSH failed login detection is a good first SIEM integration because it is simple to generate, easy to understand, and relevant for SOC/Blue Team investigations.

## Expected result

Generate a failed SSH login event, process it with Wazuh, trigger a custom alert, and document the result with screenshots.
